What's new in EaseUI
Here's what we shipped this release. As always, your feedback drove most of these improvements.
New Features
Design System Extraction
Automatically extract tokens, components, and guidelines from any generated design. Export as JSON or CSS variables.
Multi-variant Canvas
Compare up to 6 design variants side-by-side on the infinite canvas with tree-style connections.
Image-to-Design Pipeline
Upload a screenshot or mockup and let EaseUI extract the design system, recreate it, and generate code.
Improvements
→  Canvas performance improved by 40% on large projects
→  Code export now supports Tailwind v4
→  Smarter prompt understanding for component-level generation
→  Dark mode toggle for generated previews
